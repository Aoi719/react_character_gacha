# キャラクターガチャ

## ページ構成

- HOME(/)・・・トップ画面（ナビゲーション）
- GACHA(/gacha)・・・ガチャ実行ページ（ボタン）
- RESULT(/result)・・・ガチャ結果表示ページ（結果の画像）
- MyPAGE(/user)・・・所持キャラクター一覧（一覧ひょじ、フィルター検索）
- Admin(/admin)・・・管理者用ページ（キャラクターの追加・削除）

### キャラクター情報

- id （ユニーク ID）
- name （キャラクター名）
- type （属性 火、水、風、土、音）
- rarity （レアリティ N, R, SR, SSR）
- description （紹介文）
- image （画像）
- stats （能力数値。attack、defense、speed）
